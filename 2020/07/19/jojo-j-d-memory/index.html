只是自己对于迪奥的一些妄想

迪奥第一次体会到了慌乱。即便是儿时被乔纳森打哭，即便是在乔斯达邸被困在火场中，即便是最终被乔纳森打败摔下悬崖，他都不曾有过慌乱感。
他承认，乔纳森的变化是他没有料到的。这个一直处在他之下的人，却在这时一次次挡在他通往目标的路上，阻碍着他前进，甚至用人类的力量打败了他。但就算如此，他也从不觉得自己被逼到了绝境。不论乔纳森用了什么计策，他都有自信破除这个障碍。“慌乱”是只有人类拥有的多余情感，而他迪奥，是超越人类的存在，是不可能拥有这种弱点的。 可是，在那艘船上时，他却体会到一种措手不及的感觉。
“乔乔……？”没有人回应。
这时，迪奥才发觉那双抱着自己的手正逐渐变得僵硬。虽然周围的空气因爆炸变得炽热，他依然能感受到那人的身体正变得冰冷。那一瞬，就像是比赛获得了第一名，却发现只有自己一个人参赛一般，迪奥发现自己在这世上已失去了唯一能够和其战斗的对手。对于这一事实，他甚至有些无法接受。
乔纳森·乔斯达就这么死了？ 我不应该感到高兴吗？迪奥对自己心中冒出的异样情感感到不解。一切都在我的预料之中，一切都如我计划的那样发展，为什么我却没有感到“安心”？
这时，迪奥才发现他远没有自己想象的那样了解自己。但他只犹豫了一瞬间。拘泥于多余的情感并不是他的风格，即将登上世界顶点的狂喜很快冲淡了那一丝慌乱。在火光的遮蔽下，他与乔纳森的身躯融为一体，并躲进了箱子之中。出于一种奇妙的情感，他将那人的头颅也一同带入了海底。
既然如此，你就看着我支配世界吧，乔乔。

在黑暗中沉睡的日子并不好过。虽然迪奥惊讶于乔纳森身体与自己的契合度之高，但缺乏养分的摄入，伤口的愈合依旧是否缓慢。黑暗与无声总是会让人产生倦意，对于吸血鬼也是如此。迪奥已记不清自己从梦境中醒来多少次，又在黑暗之中再次沉睡。
迪奥并不认为自己是个会被过去困住的人。过去对他而言仅仅是一张写满琐事的薄纸。但是，在海底的生活中，他却常常梦到过去的日子。
“乔乔，你打算攻读哪一门课程？”
“诶？我吗？大概是考古学吧？迪奥呢？不过迪奥的话应该不管哪科都学得很好吧？”
“我大概会去读法吧。不过研究过去什么的，还真有你的风格呢乔乔……”
“……”
那是他和乔纳森共同度过的青春。
梦境逐渐变得模糊，眼前出现的仍旧是一片黑暗。
“你还真是阴魂不散啊……”迪奥感受着手中骨骼的坚硬触感，再次陷入了沉睡。

在清醒的时候，迪奥不止一次回忆起那次爆炸的情景。尽管他确信自己不会再犯同样的错误，但他仍然不明白，自己当时为什么会对乔纳森的死感到恐慌。
“奇妙的友情”，他记得乔纳森这么称呼他们之间的关系。但是迪奥从来都不觉得他们间有什么友情存在。“友情”这种东西，不过是弱者之间的抱团取暖罢了。真正的胜者只需要利用与被利用的关系。但他又是确确实实感受到了那种奇妙的情感。
也许这就是我们是命运共同体的证明吧。

深夜的开罗笼罩在一团黑暗之中。迪奥站在窗前，凝视着自己的倒影。脖颈处的伤口几乎已经愈合，只是暗红色的血痂仍显得有些恐怖。
就快了。迪奥转身斜倚在床上，继续看起丢在一旁的书。突然，他感受到一道目光正直视着他。左肩上的星形印记也微微发热，现出淡淡的红色。没过多久，热量散去，目光也随之消失，一切都恢复如常。但迪奥知道，这不是幻觉。
他合上手中的书，从旁边的柜子上拿下一颗头骨放在手上仔细端详，似乎想从那两个空洞中看到什么。
“你们乔斯达家的人都热衷于妨碍别人的事吗？”
“是时候给这段宿命画上句号了，乔乔……”

感觉虽然dio不是那种会为人类情感所困的存在，但是在他百年的人生中总会有某个时刻想起那个曾打败过他的人吧。
